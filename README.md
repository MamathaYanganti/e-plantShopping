# Paradise Nursery - Shopping Cart Application

## Overview

Paradise Nursery is an online plant shop that offers a variety of beautiful and healthy houseplants. This React and Redux-based application allows users to browse plants, add them to a shopping cart, update quantities, and manage their purchases.

## Features

### Landing Page

* Attractive landing page with a background image
* Company information and introduction
* "Get Started" button to navigate to the product listing page

### Product Listing Page

* Displays houseplants grouped into categories
* Shows plant image, name, description, and price
* Add to Cart functionality
* Dynamic cart item counter
* Disabled Add to Cart button after adding an item

### Shopping Cart Page

* Displays all items added to the cart
* Increase or decrease item quantity
* Remove items from cart
* Displays total number of items and total cost
* Continue Shopping button
* Checkout button with "Coming Soon" message

### Navigation Bar

* Links to Home, Plants, and Cart pages
* Shopping cart icon with dynamic item count

## Technologies Used

* React
* React Router
* Redux Toolkit
* JavaScript
* CSS

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/e-plantShopping.git
```

Navigate to the project folder:

```bash
cd e-plantShopping
```

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm run preview
```
